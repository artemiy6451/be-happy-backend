# Be Happy backend app
---
Бекенд часть проекта телеграм приложения Be Happy


# Install
---
1. Установить Python 3.12 и pip
2. Установить make
3. Установить poetry
```zsh
pip install poetry
```
4. Установить зависимости проекта
```zsh
make install
```
5. Заполнить .env файл
6. Создать базу данных
```zsh
make db
```

# Start
---
```zsh
make run
```

# Dev
---
```zsh
make dev
```
